# Навыки

Навыки — это инструкции, которые определяют поведение ИИ-агента. Вам не нужно
запоминать названия навыков или синтаксис — просто пишите что хотите сделать,
и агент сам выберет подходящий навык.

## Основные навыки

### /plan — Планирование задачи

**Когда:** Вы описываете задачу в свободной форме.

```
исправь баг с авторизацией
добавь кнопку "Экспорт в PDF"
отрефактори модуль оплаты
```

Агент начнёт с **фазы интервью** — задаст 3+ уточняющих вопроса о поведении, граничных случаях и ограничениях. Затем оценит сложность, создаст задачу с целью и критериями приёмки, разобьёт на шаги. Пропустить интервью можно через `--skip-interview` или подав подробную спецификацию сразу.

### /ship — Завершение

**Когда:** Работа сделана и нужно всё закрыть.

```
готово, отправляй
закрывай и сохраняй
готово, отправляй
```

Агент проверит код, прогонит тесты, убедится что критерии приёмки выполнены,
закроет задачу и предложит закоммитить. После коммита проверит структурные изменения и предложит обновить документацию. Всё в одной операции.

### /start — Начало сессии

**Когда:** Вы садитесь работать (обычно в начале дня).

```
начинай работу
start
```

Агент откроет сессию, покажет что было сделано в прошлый раз,
какие задачи ждут, предложит чем заняться.

### /end — Конец сессии

**Когда:** Закончили работу.

```
на сегодня всё
finish
```

Агент сохранит метрики, запишет передачу контекста для следующей сессии,
зафиксирует нерешённые вопросы.

## Анализ и качество

### /review — Проверка кода

```
проверь мой код
review src/auth/
```

Проверяет по 28-пунктному контрольному списку SENAR: выход за рамки задачи, фантомные импорты,
жёстко заданные значения, безопасность, тесты, граничные случаи, обратная совместимость.

### /test — Тесты

```
запусти тесты
напиши тесты для auth модуля
```

Запускает существующие тесты или пишет новые. Отслеживает покрытие.

### /explore — Исследование

```
разберись как работает их API
исследуй варианты кэширования
```

Ограниченное по времени исследование (по умолчанию 30 минут) без написания боевого кода.
Результаты фиксируются, можно создать задачу по итогам.

## Вспомогательные

### /daily — Итоги дня

```
что я сделал сегодня?
итоги
```

Показывает завершённые задачи, коммиты, метрики за день.

### /next — Следующая задача

```
что делать дальше?
следующая задача
```

Подбирает лучшую задачу для работы, показывает контекст и известные тупики.

### /commit — Коммит

```
сохрани изменения
```

Создаёт фиксацию с проверкой quality gates и области изменений.

### /checkpoint — Сохранение контекста

```
checkpoint
```

Сохраняет текущее состояние без завершения сессии. Полезно при долгой работе.

### /diff — Изменения

```
что изменилось?
покажи diff
```

Анализирует git diff с подсветкой рисков.

## Специализированные навыки

| Навык | Когда использовать |
|-------|--------------------|
| `/debug` | Систематическая отладка с поиском первопричины |
| `/security` | Аудит безопасности по OWASP Top 10 |
| `/retro` | Ретроспектива: что работало, что нет |
| `/ultra` | Глубокий анализ сложного архитектурного решения |
| `/onboard` | Быстрое знакомство с незнакомой кодовой базой |
| `/skill-test` | Автогенерация и валидация тестовых сценариев для любого навыка |

## Расширение пользовательскими навыками

Всё перечисленное выше — **встроенные** навыки: эти 34 навыка поставляются с TAUSIK и доступны сразу после начальной настройки.

Помимо встроенных, в TAUSIK есть **система установки навыков** для добавления пакетов из GitHub. Они расширяют агента: интеграция с Jira/Bitrix24, публикация в Confluence, мониторинг Sentry, пресейл-оценка и многое другое.

```bash
# Добавить TAUSIK-совместимый репозиторий навыков
.tausik/tausik skill repo add https://github.com/Kibertum/tausik-skills

# Установить навык (копирует файлы + ставит pip-зависимости)
.tausik/tausik skill install jira

# Список: активные, установленные, доступные
.tausik/tausik skill list

# Деактивировать (убрать из контекста агента, файлы остаются)
.tausik/tausik skill deactivate jira
```

Официальный репозиторий (`Kibertum/tausik-skills`) содержит 22 дополнительных навыка.

Подробнее в **[Руководство по навыкам](vendor-skills.md)** — установка и каталог. Для создания своих: **[Адаптация скиллов](skill-adaptation.md)**.
